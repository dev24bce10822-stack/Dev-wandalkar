# **Process Flow Diagrams**



Main System Workflow



Start → Initialize Data → Display Dashboard →

↓

Show Main Menu

↓

\[1] Student Management

\[2] Course Management

\[3] Enrollment Management

\[4] Search \& Reports

\[5] Exit

↓

User Selection → Execute Module → Return to Menu





Student Management Flow



Student Management → Display Options →

↓

\[1] Add Student → Input Details → Validate → Save → Confirm

\[2] View All Students → Retrieve → Display List

\[3] Search Students → Input Criteria → Search → Display Results

\[4] Back to Main Menu





Course Management Flow  



Course Management → Display Options →

↓

\[1] Add Course → Input Details → Validate → Save → Confirm

\[2] View All Courses → Retrieve → Display Catalog

\[3] Search Courses → Input Criteria → Search → Display Results

\[4] Back to Main Menu





Enrollment Management Flow



Enrollment Management → Display Options →

↓

\[1] Enroll Student → Select Student → Select Course → Set Grade → Save

\[2] Update Grade → Select Enrollment → Input New Grade → Validate → Update

\[3] View All Enrollments → Retrieve → Display List

\[4] Back to Main Menu





Search \& Reports Flow



Search \& Reports → Display Options →

↓

\[1] Search Students → By Name/Email → Execute Search → Display

\[2] Search Courses → By Name → Execute Search → Display

\[3] Student Reports → Enrollments → Grades → Analytics

\[4] Course Reports → Popularity → Enrollment Stats

\[5] Back to Main Menu
