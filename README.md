# StudentManagementSystem

A Java-based Student Management System with Spring Boot.



\## Installation \& Setup

1\. Clone the repository

2\. Run `mvn clean install`

3\. Run `mvn spring-boot:run`

4\. Access API at `https://localhost:8443/api`



\*\*Note\*\*: The application uses HTTPS. Your browser may show a security warning for the self-signed certificate. Click "Advanced" and "Proceed to localhost" to continue.
